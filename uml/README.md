## PlantUML Installation

Debian-based

```terminal
apt install plantuml
```

Arch-based

```terminal
pacman -S plantuml
```

## PlantUML Usage

### Manual generation

```terminal
plantuml *file*
```

### GUI (auto-refresh)

```terminal
plantuml -gui *dir*
```

Add the file extension if missing
